# Luogu Motaki

Luogu Motaki 致力于提供一站式的[洛谷冬日绘板](https://www.luogu.com.cn/discuss/show/288801)工具。

功能包括：

1.  图片转换：将普通的图片转化为像素点，支持缩放、裁剪、抖动色……
2.  制定计划：分图层；可以实时绘板为背景便捷地确定位置，移动图层；可以从图片转化而来，也可以手绘，还可以从云端导入/自动更新；每个图层可单独设置权重；可整体/单个图层导出……
3.  Cookies 管理：便捷地添加/删除 Cookies；为 Cookies 添加注释；统计每个 Cookie 的平均绘制速度……
4.  绘制监控：多线程异步绘制；设置绘制参数；查看当前绘板；查看绘板热度；统计绘制速度……

## Motaki？

本项目参考了 Vue.js 的命名方式（[Vue.js - The Documentary](https://www.bilibili.com/video/av92525472/) 的 6:29 处）。"motaki" 是 "painter" 的塞索托语翻译。

## Components

-   [luogu-motaki-cmd](luogu-motaki-cmd): 命令行工具。
-   [luogu-motaki-gui](luogu-motaki-gui): GUI 工具。
-   [luogu-motaki-core](luogu-motaki-core): 命令行工具和 GUI 工具共用的库。
